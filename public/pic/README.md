# 等級圖片說明

請將以下等級圖片放置在此目錄中：

- `level0.png/.jpg` - 等級 0 (0-199 分)
- `level1.png/.jpg` - 等級 1 (200-399 分) 
- `level2.png/.jpg` - 等級 2 (400-599 分)
- `level3.png/.jpg` - 等級 3 (600-799 分)
- `level4.png/.jpg` - 等級 4 (800 分以上)

## 圖片要求

- 格式：PNG 或 JPG
- 建議尺寸：100x100 像素
- 文件大小：建議小於 100KB
- 風格：適合小學生的可愛卡通風格

## 等級系統

- 每 200 分升一級
- 等級上限為 4 級
- 經驗條會顯示到下一級所需的進度
